Sistema integral de gestión de normativa universitaria con generación automática de PDFs.

== Description ==

Plugin profesional para la gestión completa de normativa institucional.

= Características =

* Custom Post Type con campos completos
* Control de versiones
* Relaciones entre normas
* Importación/Exportación CSV, JSON, XML
* Sanitización HTML configurable
* Buscador avanzado
* Múltiples modos de visualización
* Diseño responsive

= Shortcodes =

* `[ull_normativa_listado]` - Listado con filtros
* `[ull_normativa_buscador]` - Buscador avanzado
* `[ull_norma id="123"]` - Ficha individual
* `[ull_normativa_archivo]` - Archivo por años

== Installation ==

1. Sube `ull-normativa` a `/wp-content/plugins/`
2. Activa el plugin
3. Configura en 'Normativa > Configuración'
